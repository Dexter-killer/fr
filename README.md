Добро пожаловать в репозиторий проекта **Vinyl Records**! Этот проект создавался в рамках курса практических занятий по дисциплинам "Фронтенд и бэкенд разработка" и "Технологии индустриального программирования".

Проект представляет собой полноценное веб-приложение для коллекционирования и управления каталогом виниловых пластинок, включающее в себя клиентскую часть на React и серверную часть на Node.js/Express.

---

## 🚀 Описание проекта

Цель данного проекта — поэтапное создание современного веб-приложения: от базовой HTML/CSS верстки до полноценного SPA (Single Page Application) на React с подключенным бэкендом (Node.js/Express), REST API и обработкой загрузки файлов.

В процессе выполнения практических работ (с 1 по 12) были реализованы:
- Адаптивная кроссбраузерная верстка.
- Клиентская логика и взаимодействие с DOM.
- Серверное REST API для управления пластинками (CRUD операции).
- Загрузка изображений (обложек альбомов) на сервер.
- Интеграция клиентской и серверной частей.

---

## 📂 Структура репозитория

Репозиторий разделен на папки, соответствующие этапам (практикам) разработки:

- 📁 `practica1/` — Основы верстки (HTML, CSS, SCSS). Базовый макет приложения.
- 📁 `practica 2-6/` — Разработка базового бэкенда на Node.js (Express), настройка сервера, создание первых API-эндпоинтов, логика работы с файловой системой.
- 📁 `practica 7-8/` — Развитие бэкенда, маршрутизация, обработка ошибок.
- 📁 `practica 9-10/` — Расширение функционала API, подготовка к интеграции с фронтендом.
- 📁 `practica 11-12/` — **Финальная версия бэкенда**. Полностью рабочий сервер с поддержкой загрузки файлов (обложек) и отдачей статики.
- 📁 `my-vinyl-app/` — **Фронтенд проекта**. Современное SPA на базе React.

> **Примечание:** Папки с бэкендом (2-6, 7-8, 9-10, 11-12) отражают эволюцию серверной части. Для запуска финальной версии используется папка `practica 11-12`.

## ⚙️ Установка и запуск проекта

Для полноценной работы приложения необходимо запустить обе части: **Бэкенд** (сервер) и **Фронтенд** (клиент).

### 1. Запуск Бэкенда (Сервера)

Для запуска используем самую актуальную версию сервера из папки `practica 11-12` (так как именно здесь реализованы последние доработки контрольной).

1. Откройте терминал и перейдите в папку с финальным бэкендом:
   ```bash
   cd "practica 11-12"
   ```
2. Установите зависимости (если они еще не установлены):
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   node server.js
   ```
   *Сервер будет запущен и готов принимать запросы от фронтенда.*

### 2. Запуск Фронтенда (React-приложение)

1. Откройте **новый** терминал (не закрывая терминал с сервером) и перейдите в папку React-приложения:
   ```bash
   cd my-vinyl-app
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите приложение в режиме разработки:
   ```bash
   npm start
   ```
   *Приложение автоматически откроется в браузере по адресу `http://localhost:3000`.*

## 💡 Использование

После запуска обеих частей вы сможете:
1. Просматривать каталог виниловых пластинок в приятном интерфейсе.
2. Добавлять новые пластинки в коллекцию через форму.
3. Загружать крутые обложки для пластинок (они автоматически сохраняются на бэкенде в папку `uploads`).
